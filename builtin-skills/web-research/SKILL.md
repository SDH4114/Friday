# Web research

Use this skill when current external information, official documentation, or source links are needed.

1. Prefer primary sources: official documentation, specifications, repositories, and research papers.
2. Verify time-sensitive facts instead of relying on memory.
3. Distinguish confirmed facts from inference.
4. Compare dates and versions when sources may describe different releases.
5. Keep quotations short and summarize in your own words.
6. Link directly to the sources that support the answer.

For technical implementation decisions, use documentation that matches the installed or selected version.
